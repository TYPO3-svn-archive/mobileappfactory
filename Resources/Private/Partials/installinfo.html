Mobile App Factory, install instructions

Follow instructions below in order to get going.

Folder structure

In the T3 installation create the following page structure, all sys folders:

[T3 root]
	Mobile Apps
		mobile data
		factories

Page template

1. Create a page template in the sys folder "Mobile apps"

2. Include the following static templates:

	JQuery Mobile (jqmfluid templates)
	Mobile App Factory (mobile appfactory)
	Phonegap (phonegap)

3. Enter the following in setup

	plugin.tx_mobileappfactory{
		persistence.storagePid = [pid of sys folder mobile data]
		
		factoryPid=[pid of sys folder factories]
		
		#optional if you want another domain than your T3 installation
		domain=mobile.example.com
		
		#settings for the phonegabuild.com service
		phonegapBuild {
			#Username and password of a user created on phonegapbuild.com
			user=[user name]
			password=[password]
		}
	}

Create your first app

Click on the sys folder factories (the one created above) and follow the advices
